# LifeRPG

LifeRPG is a gamified productivity and habit-tracking application inspired by RPG progression systems.

Users can:
- Create tasks
- Complete daily challenges
- Earn Life Points (LP)
- Level up characters
- Track progression through RPG-style mechanics

## Tech Stack

Frontend:
- Angular
- TypeScript
- Tailwind CSS

Backend:
- ASP.NET Core Web API
- Entity Framework Core

Database:
- SQL Server

## Status

🚧 Under active development.

Current features:
- User registration/login
- Task management
- Daily tasks
- Character progression
- XP and LP system

Planned features:
- Achievements
- Character unlock system improvements
- Better statistics and analytics
- Mobile-friendly UI
